# Install Context7 MCP

```bash
pnpm mcp:install context7
# opcional: pnpm mcp:install context7 -- --context7-key SUA_KEY
```

Depois: **Settings → Tools & MCP** → Reload Window.

Use para docs versionadas de Next/React/libs em vez de chutar APIs.
