# DRY — checklist consolidado

## Conhecimento

- [ ] Regra de negócio SSOT
- [ ] Schema → type infer
- [ ] Constants centralizadas

## Duplicação consciente

- [ ] Duplicar quando domínios distintos
- [ ] Rule of three antes extract
- [ ] Documentado se intencional

## Abstração

- [ ] ≥3 usos antes framework
- [ ] API mais simples que inline
- [ ] Remove dead abstraction

## Anti-padrões

- Utils junk drawer
- DRY mecânico
- BaseClass prematura
