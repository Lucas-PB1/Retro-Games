# UX — checklist consolidado

## Tarefa

- [ ] Objetivo claro por tela
- [ ] Fricção mínima
- [ ] Empty state útil

## Heurísticas

- [ ] Status visível
- [ ] Ação primária única
- [ ] Undo/back disponível

## Erros

- [ ] Mensagem acionável
- [ ] Campo associado
- [ ] Loading anti double-submit

## A11y uso

- [ ] Keyboard completo
- [ ] Contraste AA
- [ ] Não só cor

## Anti-padrões

- Dark patterns
- Modal chains
- Jargão técnico na UI
