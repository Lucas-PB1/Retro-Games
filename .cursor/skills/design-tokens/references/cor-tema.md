# Cor e tema

## Semântica de cor

| Papel | Uso |
|---|---|
| bg / surface | fundos |
| text / icon | conteúdo |
| border | divisores |
| action / accent | CTAs |
| danger / warning / success | feedback |

## Tema claro/escuro

- Troque **aliases semânticos**, não componentes
- Contraste AA em ambos
- Não invente dark mode se o produto não tem

- [ ] Pares texto/fundo com contraste ok
- [ ] Estado não só por cor (ícone/texto)
- [ ] `color-scheme` / classe de tema alinhada ao repo
