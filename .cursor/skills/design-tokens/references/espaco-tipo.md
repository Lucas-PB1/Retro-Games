# Espaço, tipografia, raio

## Espaço

Escala consistente (4/8 ou a do DS). Densidade de app ≠ marketing.

## Tipografia

- Escala limitada (display / title / body / caption)
- Line-height e measure legíveis
- Peso para ênfase, não para ruído

## Raio / sombra

Poucos degraus (`sm/md/lg`). Sombra = elevação semântica, não decoração aleatória.
