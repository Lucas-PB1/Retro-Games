# Clean Code — checklist consolidado

## Nomes

- [ ] Intenção clara; boolean prefix
- [ ] Vocabulário consistente
- [ ] Sem magic numbers

## Funções

- [ ] Pequena; um nível abstração
- [ ] Params object se >3
- [ ] Command/query separados

## Comentários

- [ ] Why not what
- [ ] Sem código morto

## Smells

- [ ] Duplicação endereçada
- [ ] Refactor incremental com testes

## Anti-padrões

- Clever one-liner
- God function
- Commented code

## Skills relacionadas

| Tópico | Skill |
|---|---|
| DRY | `dry` |
| SOLID | `solid` |
