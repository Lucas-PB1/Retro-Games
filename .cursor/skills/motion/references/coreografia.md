# Coreografia

## Entrada / saída

- Entrada: fade + leve translate (ease-out)
- Saída: mais rápida que entrada (não prender o usuário)
- Stagger curto em listas (≤50ms entre itens; limite itens)

## Sequência

1. Elemento focal primeiro
2. Secundários depois
3. Decoração por último — ou nunca

## Shared element / continuidade

Se o projeto já tem padrão de “mesmo objeto atravessa telas”, preserve; senão não invente.

## Checklist

- [ ] Ordem de animação = ordem de importância
- [ ] Saída não bloqueia navegação
- [ ] Stagger não atrasa conteúdo crítico
