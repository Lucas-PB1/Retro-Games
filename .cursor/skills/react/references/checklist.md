# React — checklist consolidado

## Componentes

- [ ] Responsabilidade única
- [ ] Props tipadas (TS)
- [ ] Composição via children/slots

## Estado

- [ ] Mínimo; derivados no render
- [ ] Server state via query/RSC
- [ ] Sem mutação direta

## Effects

- [ ] Só sincronização externa
- [ ] Cleanup presente
- [ ] Deps corretas

## Listas

- [ ] Keys estáveis (id)
- [ ] Virtualização se necessário

## Anti-padrões

- Effect para derived state
- Context god object
- Index keys em lista dinâmica

| Tópico | Skill |
|---|---|
| Next.js | `nextjs` |
| TypeScript | `typescript` |
| UI/UX | `ui`, `ux` |
